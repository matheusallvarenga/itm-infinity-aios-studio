# itm-infinity-aios-studio

> AIOS Studio — Visual interface + headless SDK for the AIOS framework.

## Overview

Dashboard, Kanban, Agents Explorer, Squads Browser, Workflow Runner for the AIOS framework.
Integrates with [itm-legendary-infinity-os](https://github.com/matheusallvarenga/itm-legendary-infinity-os) as a studio submodule, AND can be consumed programmatically via SDK.

## Installation

```bash
npm install itm-infinity-aios-studio
```
